• Assign 3 dots to traits.
  • Fill in ability scores based on chosen traits.
  • Create skills for each ability.
  • Select 1 skill to be a focus skill.
  • Select a combination of up to 3 perks and/or penalties.
  • Set starting Doom to 1 + number of perks.
  • Set starting Destiny to 1 + number of penalties.
  • Assign 180 points to skills with the following two restrictions:
    • No skill under an ability can have more points than the ability.
    • Must spend at least as many points on skills under an ability as their are points in the ability.
